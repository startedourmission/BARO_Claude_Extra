# Computer Use로 카카오톡 자동화하기

> 공식 API가 없는 카카오톡을, 클로드가 화면을 직접 보고 조작하게 해서 자동화합니다. 부재중 답장부터 가족 공지, 중요 메시지 요약까지.

## 들어가며

한국에서 일하거나 사는 사람에게 카카오톡은 "통신수단"을 넘어 **일상 인프라**입니다. 그런데 이걸 자동화하려고 하면 벽에 부딪힙니다.

- **일반 사용자용 공식 API 없음** — 비즈니스/알림톡 API는 사업자 등록과 심사 필요
- **봇 계정 금지** — 카카오 ToS 위반
- **브라우저 크롤링도 막힘** — 2단계 인증과 기기 등록
- **MacOS용 kakaocli 같은 도구들** — 텍스트 읽기·전송은 되지만 미디어·이모티콘·스크롤 UI는 한계

그래서 오랫동안 "카톡을 AI와 엮어 쓰는" 건 사실상 포기 영역이었습니다. 그런데 2024년 말 공개된 **Computer Use**가 판을 바꿨습니다. 클로드가 **내 화면을 직접 보고, 마우스·키보드를 조작**하므로, API가 없는 앱도 **사람이 쓰는 것과 똑같이** 쓰게 할 수 있습니다.

이 부록에서는 Computer Use를 카카오톡에 붙여 현실적으로 쓸모 있는 자동화를 만드는 방법을 정리합니다. 동시에 **ToS와 상식의 경계**도 분명히 짚습니다.

## 먼저, 경계 긋기

시작 전에 반드시 확인할 것.

1. **내 계정으로, 내 화면에서만** — 남의 계정 자동화는 카카오 ToS 위반이자 범죄 소지
2. **스팸 금지** — 같은 메시지를 여러 방에 일괄 발송하는 "공지" 용도는 괜찮지만, 대량 홍보·알림은 비즈메시지 API로 가야 합니다
3. **민감 대화 자동 응답 금지** — 금융·의료·법률 관련 대화에 LLM이 대신 답장하게 두지 말 것
4. **상대방 동의** — "AI가 대신 답장한다"는 사실을 알 권리가 있는 관계(가족·친한 친구 제외)라면 미리 고지

이 네 가지를 지킨다면, **개인 생산성 도구**로서 카톡 자동화는 합리적입니다. 이 글의 예시들도 전부 이 선 안에서 설계됩니다.

## Computer Use의 역할

Computer Use는 Anthropic이 제공하는 "**화면을 보는 눈 + 누르는 손**" 모드입니다. 클로드가 할 수 있는 일:

- 스크린샷을 찍어 현재 화면 상태 파악
- 마우스 이동·클릭·드래그
- 키보드 입력
- 스크롤
- 앱 전환

카카오톡 데스크톱 앱을 **사람이 쓰는 그대로** 조작합니다. 메시지 읽기·답장·검색·파일 첨부·이모티콘 선택 전부 가능합니다.

### 기존 도구와의 관계

| 도구 | 잘하는 것 | 못하는 것 |
|------|-----------|-----------|
| `kakaocli` (Homebrew tap) | 텍스트 메시지 읽기·검색·전송 (빠름) | 이모티콘·파일·복잡 UI |
| Computer Use | 화면에 있는 건 뭐든 조작 | 느림, 토큰 소비 큼 |
| 카카오 비즈API | 공식, 안정적 | 사업자 등록 필수, 개인 불가 |

결론: **일상 텍스트 전송은 kakaocli**, **UI 조작이 필요한 작업은 Computer Use**, **상업용 대량 발송은 비즈API**. 개인 자동화는 앞의 두 개 조합으로 거의 다 됩니다.

## 준비 단계

### 1. 카카오톡 Mac 앱 설치

App Store에서 "카카오톡" 설치 후 로그인. PC/모바일 모두에서 이중 로그인 유지.

### 2. macOS 권한 부여

Computer Use는 두 가지 시스템 권한이 필요합니다.

```
시스템 설정 → 개인정보 보호 및 보안
  ├─ 화면 녹화 → [터미널 / 클로드 데스크톱] 체크
  └─ 손쉬운 사용 → [터미널 / 클로드 데스크톱] 체크
```

### 3. Claude Code 또는 Claude Desktop에서 Computer Use 활성화

**Claude Code:**
```bash
claude --computer-use
# 또는 세션 중
/enable-computer-use
```

**Claude Desktop:** 설정 → Features → Computer Use 토글.

### 4. kakaocli 병행 설치 (선택, 강력 추천)

```bash
brew install silver-flight-group/tap/kakaocli
```

텍스트만 다루는 작업은 kakaocli가 훨씬 빠릅니다. Computer Use를 **보조**로 두는 구성이 현실적입니다.

### 5. 전용 작업 데스크톱 만들기

macOS Mission Control에서 **데스크톱 3**을 만들어 "카카오톡 자동화 전용"으로 지정하세요. 자동화 중 다른 앱이 끼어들지 않도록 격리합니다.

## 실전 시나리오

### 시나리오 1 — 부재중 지능형 답장

회의 중·운전 중에 카톡이 왔을 때 "지금 운전 중입니다" 같은 획일적 답장 대신, **상황을 파악해 적절한 응대**를 합니다.

```text
> 카카오톡을 모니터링해 줘.
> 새 메시지가 오면:
>   1. 발신자 이름 확인
>   2. 메시지 내용 읽기
>   3. 아래 기준으로 판단:
>      - 가족/회사 긴급 → 알림만 띄우고 답장 X
>      - 일반 지인 → "회의 중이라 나중에 답장할게요" + 상대방 메시지 요약을 별도 메모로
>      - 광고/영업 → 무시
> 절대 금융·계좌·비밀번호 관련 대화에는 응답하지 말 것
```

클로드가 카톡 화면을 주기적으로 캡처해, 새 메시지를 읽고 분류합니다. 답장을 보내기 전엔 **반드시 확인을 구하는 프롬프트**를 띄우게 하세요. 자동 발송은 사고의 지름길입니다.

### 시나리오 2 — 가족 그룹방 공지 루틴

부모님·형제 그룹방에 "아빠 퇴근 중", "오늘 저녁 7시에 같이 먹자" 같은 일상 공지를 **반복 없이 보내기**.

```text
> 캘린더에서 오늘 저녁 가족 약속 있는지 확인해 줘.
> 있으면 "가족" 그룹방에 한 줄 공지 작성 후,
> 보내기 전 내 확인을 받고 전송해 줘.
> 이모티콘 하나 자연스럽게 붙여서.
```

- 캘린더 읽기는 gws-cli MCP로
- 공지 작성은 클로드가
- 전송은 Computer Use로 카톡 앱 조작
- 확인은 데스크톱 알림 or 터미널 프롬프트

**주의:** "자동 발송" 모드는 처음엔 끄고, 익숙해진 후에도 가족방 같은 안전한 곳에만 허용.

### 시나리오 3 — 쌓인 메시지 일괄 요약

외부 일정 길었던 날, 읽지 않은 대화방 50개 열어보는 피로를 없앱니다.

```text
> 카카오톡에서 읽지 않은 메시지 있는 방 전부 확인해 줘.
> 각 방별로 마지막 10개 메시지만 빠르게 훑어서,
> 아래처럼 요약 브리핑 만들어줘:
>   - 긴급 답장 필요: [방 이름, 한 줄 요약]
>   - 정보성/확인만: [방 이름]
>   - 스팸/단체 광고: [방 이름]
> 절대 메시지를 보내지는 말 것. 요약만.
```

결과를 마크다운으로 받으면 5분 안에 "오늘의 카톡 인박스"가 정리됩니다.

### 시나리오 4 — 중요 메시지 감지 알림

회의 중에는 카톡을 닫아두는데, **진짜 중요한 메시지**는 놓치고 싶지 않다면.

```text
> 백그라운드에서 카톡을 5분마다 체크해 줘.
> "엄마", "응급", "사고", "병원" 같은 키워드나,
> 배우자/직속 상사로부터 온 메시지가 있으면
> 즉시 macOS 알림으로 띄워줘.
> 나머지는 무시.
```

평상시 우리 주의력을 훔쳐 가는 카톡을 "진짜 중요한 건만 알림 오는" 도구로 재정의합니다.

### 시나리오 5 — 약속 자동 정리

약속 잡는 대화에서 날짜·장소가 확정되면 캘린더에 자동 등록.

```text
> 지금 열려 있는 대화방에서 최근 20개 메시지 읽고,
> 만날 약속이 확정되었는지 판단해 줘.
> 확정이면:
>   - 날짜, 시간, 장소, 참석자 추출
>   - 구글 캘린더에 이벤트 생성 (gws-cli로)
>   - 생성 전 내 확인
> 모호하면 "확인 필요"로 표시하고 넘어가.
```

카톡 + 캘린더 사이의 수동 복붙이 사라집니다.

## 작업 레시피 — "확인 후 전송" 패턴

Computer Use로 메시지를 보낼 때 **반드시 지켜야 할 골든 룰**: 전송 전 인간 확인. 아래 템플릿을 SKILL.md로 저장해 두세요.

```markdown
---
name: kakao-send
description: 카카오톡에서 메시지 보내기 전 확인 단계 필수
disable-model-invocation: true
---

카카오톡 메시지 전송 절차:

1. 받는 사람(방 이름) 명확히 확인
2. 메시지 내용 최종 초안을 사용자에게 보여주기
3. 사용자가 "보내", "전송", "OK" 중 하나 답할 때만 실제 전송
4. 전송 후 스크린샷으로 발송 확인
5. 수정·삭제 필요하면 즉시 롤백 시도

다음 경우엔 자동으로 중단:
- 받는 사람이 모호한 경우
- 금액·계좌번호·비밀번호가 포함된 메시지
- 받는 사람이 연락처에 없는 낯선 번호
```

## kakaocli와 하이브리드 구성

Computer Use는 느립니다(한 액션당 2~5초). 텍스트만 다루는 작업은 kakaocli가 수십 배 빠릅니다.

```bash
# 텍스트 읽기/검색/전송 — kakaocli로
kakaocli messages list --room "가족" --limit 20
kakaocli messages send --room "가족" --text "퇴근 중"
kakaocli rooms search "프로젝트"
```

클로드가 도구를 **자동 선택**하게 하려면 CLAUDE.md에 명시하세요.

```markdown
# 카카오톡 자동화 규칙

- 텍스트 읽기·검색·전송: kakaocli를 우선 사용 (빠름)
- 이모티콘·파일·스크롤·스타일 있는 UI: Computer Use 사용
- 두 도구 모두 전송 전 확인 필수
```

## 한계와 주의점

### 속도

Computer Use의 한 액션은 2~5초. 100개 방 스캔하려면 실질적으로 몇 분 걸립니다. 배경으로 돌리세요.

### 비용

스크린샷 + 추론이 반복되므로 토큰 소비가 큽니다. 일반 텍스트 작업의 5~10배 수준. 세션당 예산을 정해 두세요.

### UI 변경에 취약

카톡 Mac 앱 업데이트 후 레이아웃이 바뀌면 클로드가 헷갈릴 수 있습니다. 접근성 트리를 쓰는 kakaocli가 더 튼튼한 이유입니다.

### 화면 잠금·절전

자동화 중 맥이 잠기거나 절전에 들어가면 멈춥니다. Automation 실행 중엔 [caffeinate](https://ss64.com/mac/caffeinate.html) 또는 Amphetamine으로 유지.

```bash
caffeinate -dims &  # 디스플레이·시스템 슬립 방지
```

### 알림 숨김 설정

카톡 앱의 "미리보기 숨김" 기능이 켜져 있으면 Computer Use가 메시지 내용을 못 읽습니다. 자동화 목적이라면 임시로 미리보기 켜두거나, 카톡 설정을 프로파일로 전환하세요.

### 2단계 인증과 기기 인증

새 기기·IP에서 로그인 시 인증 푸시가 발생합니다. 자동화 중간에 인증 화면이 뜨면 자동화가 막힙니다. 이럴 땐 사람이 개입해 풀어 주어야 합니다.

### 카카오 정책 변화

카카오는 비즈니스용이 아닌 PC 버전 자동화에 대해 공식 입장이 없지만, **남용 시 계정 제재 가능성**은 상존합니다. 상업 목적이라면 비즈메시지 API로 이전하세요.

## 윤리적 디폴트 — AI로 관계를 대체하지 말 것

자동화가 편리하지만, 기억해 둘 선.

- **친밀한 관계의 답장까지 AI에게 맡기면 그 관계 자체가 얕아집니다.** "늦게라도 내가 직접 답하기"가 기본값이어야 합니다.
- **요약·정리·필터링은 자동화**, **공감·의사결정·약속은 사람**. 이 경계를 지키면 도구가 관계를 갉아먹지 않습니다.
- 친구에게 "AI가 대신 답장하는" 사실을 들켰을 때 관계가 상처받을지 미리 상상해 보세요.

## 마치며

카카오톡은 한국에서 가장 많이 쓰는 앱이지만, 가장 자동화하기 까다로운 앱이기도 했습니다. Computer Use가 그 벽을 부분적으로 허물었습니다. 완벽하지 않고, 빠르지도 않고, ToS 경계선에서 조심스럽게 써야 하는 도구지만, **내 일상의 반복 노동**을 확실히 줄여 줍니다.

오늘 저녁, 가족 그룹방에 "저녁 먹으러 가자" 한 줄을 Computer Use로 보내 보는 것부터 시작해 보세요. 확인 단계를 반드시 거치고, 한 번에 한 방만. 그 작은 시작이, 일주일 뒤엔 "쌓인 카톡 전체를 5분에 정리하는" 루틴으로 자라 있을 겁니다.

```bash
# 가장 빠른 시작
1. brew install silver-flight-group/tap/kakaocli
2. 시스템 설정 → 화면 녹화·손쉬운 사용 권한 부여
3. claude --computer-use
4. "오늘 카톡 요약해 줘 (보내지는 말고)"
```
