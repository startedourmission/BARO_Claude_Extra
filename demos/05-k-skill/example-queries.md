# k-skill 예시 쿼리

설치 후 클로드 코드에서 자연어로 그냥 쓰면 됩니다.

## 교통

```text
다음 주 월요일 아침 서울→부산 SRT 예매해 줘. 09시 이전 출발, 특실 우선.
이 송장번호 123456789012 어디쯤 와 있어?
지금 위치 기준으로 가장 싼 주유소 3곳 알려줘.
```

## 쇼핑

```text
쿠팡에서 무선 청소기 30만원 이하 로켓배송 되는 것만 비교해 줘.
올리브영 근처 매장에서 선크림 재고 있는 거 찾아줘.
```

## 스포츠·엔터

```text
최근 KBO 5경기 결과 정리해 주고, 기아 타이거즈 경기만 따로 보여줘.
이번 주 로또 당첨번호 확인하고 내 번호 7 14 22 31 35 42와 대조해 줘.
```

## 정보

```text
해고당했을 때 실업급여 조건 법령 근거 조항으로 찾아줘.
오늘 서울 날씨 예보 알려줘.
```

## Node/Python 런타임 사전 설치 (일부 스킬 필요)

```bash
# Node 계열
npm install -g kbo-game k-lotto coupang-product-search

# Python 계열 (SRT/KTX 예매 시)
pip install SRTrain korail2 pycryptodome
```
