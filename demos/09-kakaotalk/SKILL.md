# 카카오톡 메시지 전송 절차

## 전송 순서 (반드시 준수)

1. 받는 사람(방 이름) 명확히 확인
2. 메시지 내용 최종 초안을 사용자에게 보여주기
3. 사용자가 "보내", "전송", "OK" 중 하나 답할 때만 실제 전송
4. 전송 후 스크린샷으로 발송 확인
5. 수정·삭제 필요하면 즉시 롤백 시도

## 자동 중단 조건

다음 경우엔 전송하지 않고 사용자에게 묻는다:
- 받는 사람이 모호한 경우
- 금액·계좌번호·비밀번호가 포함된 메시지
- 연락처에 없는 낯선 번호

## 도구 선택 기준

- 텍스트 읽기·검색·전송: kakaocli 우선 (빠름)
- 이모티콘·파일·스크롤·복잡 UI: Computer Use 사용
